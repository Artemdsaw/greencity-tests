# GreenCity Events – Тестування

Цей репозиторій містить прості тест-кейси для перевірки сторінки подій GreenCity.  
Мета – перевірити, що сторінка працює як треба, і всі основні елементи відображаються правильно.

## 🔗 Сторінка для тестування
[GreenCity Events](https://www.greencity.cx.ua/#/greenCity/events)

## Що протестовано
- Відображення списку подій
- Перехід на детальну сторінку події
- Роботу кнопок і посилань на сторінці
